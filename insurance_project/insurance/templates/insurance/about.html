O aplikaci

Úvod

Tato aplikace slouží k evidenci pojištěnců a jejich pojištění. Poskytuje funkce pro správu pojištěnců, včetně vytváření, úprav a mazání pojištěnců a jejich pojištění.

Testování aplikace

Pokud si chcete aplikaci vyzkoušet, můžete se přihlásit pomocí následujících účtů:

  • Admin účet: admin, Heslo: adminheslo
  • Uživatelský účet: milda_123, Heslo: poradneheslo123*

Pomocí těchto účtů si můžete vyzkoušet funkce aplikace. Admin účet má přístup ke všem funkcím správy pojištěnců a pojištění, zatímco uživatelský účet má omezená práva.

Hlavní funkce

  • Kompletní správa pojištěnců (CRUD operace).
  • Kompletní správa pojištění.
  • Autentizace a registrace uživatelů.
  • Ochrana citlivých údajů prostřednictvím autentizace.
  • Responzivní design, který zajišťuje, že aplikace vypadá dobře na různých zařízeních (časem).

Technologie

Při vývoji této aplikace jsem použila následující technologie:

  • Django - webový framework pro vývoj aplikací v Pythonu.
  • SQLite - lehká databáze pro ukládání dat.
  • HTML, CSS - pro tvorbu uživatelského rozhraní.
  • JavaScript - pro validaci formulářů na straně klienta a vylepšení interaktivity aplikace.
